# SuiteSparse Benchmark Suite

[![CI](https://github.com/fatunkaz/suitesparse-benchmarks/actions/workflows/ci.yml/badge.svg)](https://github.com/fatunkaz/suitesparse-benchmarks/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Reproducible benchmark suite for measuring and comparing the performance of
[SuiteSparse](https://github.com/DrTimothyAldenDavis/SuiteSparse) sparse linear
algebra kernels across three hardware architectures: **x86-64** (native),
**RISC-V** (via QEMU), and **AArch64** (via QEMU).

Benchmarks reproduce real usage scenarios from four applications:
[Ceres Solver](https://ceres-solver.org/),
[SUNDIALS](https://sundials.readthedocs.io/),
[SciPy](https://scipy.org/), and
[NetworkX](https://networkx.org/).

## Benchmarked scenarios

| Benchmark | SuiteSparse module | Application | Scenario |
|---|---|---|---|
| `bench_cholmod` | CHOLMOD | Ceres Solver | Repeated Cholesky factorization |
| `bench_klu` | KLU | SUNDIALS | BTF symbolic + repeated refactorization |
| `bench_umfpack` | UMFPACK | SciPy | Multifrontal LU (`spsolve`) |
| `bench_graphblas` | GraphBLAS | NetworkX | SpGEMM A×A on citation graph |

## Repository structure

```
suitesparse-benchmarks/
├── benchmarks/
│   ├── cholmod/        # CHOLMOD benchmark (Ceres Solver scenario)
│   ├── klu/            # KLU benchmark (SUNDIALS scenario)
│   ├── umfpack/        # UMFPACK benchmark (SciPy scenario)
│   └── graphblas/      # GraphBLAS benchmark (NetworkX scenario)
├── matrices/           # Test matrices (not tracked, download manually)
├── scripts/
│   └── run_all.sh      # Build and run all benchmarks
└── results/            # Benchmark output (not tracked, generated by script)
```

## Requirements

**Host:** ONLY FOR x86-64 machine running Linux (Ubuntu 24.04 recommended)

### System packages

```bash
sudo apt install -y build-essential cmake git wget m4 \
    gcc-riscv64-linux-gnu g++-riscv64-linux-gnu gfortran-riscv64-linux-gnu \
    gcc-aarch64-linux-gnu g++-aarch64-linux-gnu gfortran-aarch64-linux-gnu \
    qemu-user-static libsuitesparse-dev
```

### Required tools

| Tool | Version | Purpose |
|---|---|---|
| GCC / G++ | >= 13 | C/C++ compiler |
| CMake | >= 3.16 | Build system |
| gfortran | >= 13 | Required by OpenBLAS |
| wget | any | Downloading sources |
| m4 | any | Required by GMP |
| qemu-user-static | >= 8.0 | RISC-V and AArch64 emulation |

## Setup

### Step 1 — Build OpenBLAS for RISC-V

```bash
git clone https://github.com/xianyi/OpenBLAS.git
cd OpenBLAS
make TARGET=RISCV64_GENERIC HOSTCC=gcc \
    CC=riscv64-linux-gnu-gcc \
    FC=riscv64-linux-gnu-gfortran \
    -j$(nproc)
make TARGET=RISCV64_GENERIC PREFIX=$HOME/riscv-libs install
cd ..
```

### Step 2 — Build OpenBLAS for AArch64

```bash
cd OpenBLAS && make clean
make TARGET=ARMV8 HOSTCC=gcc \
    CC=aarch64-linux-gnu-gcc \
    FC=aarch64-linux-gnu-gfortran \
    -j$(nproc)
make TARGET=ARMV8 PREFIX=$HOME/arm64-libs install
cd ..
```

### Step 3 — Build GMP and MPFR for RISC-V

```bash
wget https://ftp.gnu.org/gnu/gmp/gmp-6.3.0.tar.xz
tar xf gmp-6.3.0.tar.xz && cd gmp-6.3.0
mkdir build-riscv && cd build-riscv
../configure --host=riscv64-linux-gnu \
    --prefix=$HOME/riscv-libs --disable-shared CFLAGS="-fPIC"
make -j$(nproc) && make install && cd ../..

wget https://ftp.gnu.org/gnu/mpfr/mpfr-4.2.1.tar.gz
tar xf mpfr-4.2.1.tar.gz && cd mpfr-4.2.1
mkdir build-riscv && cd build-riscv
../configure --host=riscv64-linux-gnu \
    --prefix=$HOME/riscv-libs \
    --with-gmp=$HOME/riscv-libs --disable-shared
make -j$(nproc) && make install && cd ../..
```

### Step 4 — Build GMP and MPFR for AArch64

```bash
cd gmp-6.3.0
mkdir build-arm64 && cd build-arm64
../configure --host=aarch64-linux-gnu \
    --prefix=$HOME/arm64-libs --disable-shared CFLAGS="-fPIC"
make -j$(nproc) && make install && cd ../..

cd mpfr-4.2.1
mkdir build-arm64 && cd build-arm64
../configure --host=aarch64-linux-gnu \
    --prefix=$HOME/arm64-libs \
    --with-gmp=$HOME/arm64-libs --disable-shared
make -j$(nproc) && make install && cd ../..
```

### Step 5 — Build SuiteSparse for RISC-V

Create `riscv64-toolchain.cmake`:
```cmake
set(CMAKE_SYSTEM_NAME Linux)
set(CMAKE_SYSTEM_PROCESSOR riscv64)
set(CMAKE_C_COMPILER riscv64-linux-gnu-gcc)
set(CMAKE_CXX_COMPILER riscv64-linux-gnu-g++)
set(CMAKE_Fortran_COMPILER riscv64-linux-gnu-gfortran)
set(CMAKE_FIND_ROOT_PATH $ENV{HOME}/riscv-libs /usr/riscv64-linux-gnu)
set(CMAKE_FIND_ROOT_PATH_MODE_PROGRAM NEVER)
set(CMAKE_FIND_ROOT_PATH_MODE_LIBRARY ONLY)
set(CMAKE_FIND_ROOT_PATH_MODE_INCLUDE ONLY)
```

```bash
git clone https://github.com/DrTimothyAldenDavis/SuiteSparse.git
cd SuiteSparse && mkdir build_riscv && cd build_riscv
cmake .. \
    -DCMAKE_TOOLCHAIN_FILE=../riscv64-toolchain.cmake \
    -DCMAKE_INSTALL_PREFIX=$HOME/riscv-libs \
    -DBLA_VENDOR=OpenBLAS \
    -DBLAS_LIBRARIES=$HOME/riscv-libs/lib/libopenblas.so \
    -DLAPACK_LIBRARIES=$HOME/riscv-libs/lib/libopenblas.so \
    -DSUITESPARSE_ENABLE_CUDA=OFF \
    -DSUITESPARSE_USE_OPENMP=OFF \
    -DGMP_INCLUDE_DIR=$HOME/riscv-libs/include \
    -DGMP_LIBRARY=$HOME/riscv-libs/lib/libgmp.a \
    -DGMP_STATIC=$HOME/riscv-libs/lib/libgmp.a \
    -DMPFR_INCLUDE_DIR=$HOME/riscv-libs/include \
    -DMPFR_LIBRARY=$HOME/riscv-libs/lib/libmpfr.a \
    -DMPFR_STATIC=$HOME/riscv-libs/lib/libmpfr.a \
    -DSUITESPARSE_ENABLE_PROJECTS="suitesparse_config;amd;btf;camd;ccolamd;colamd;cholmod;cxsparse;klu;umfpack;spqr;graphblas;lagraph"
make -j$(nproc) && make install && cd ../..
```

    -DGMP_STATIC=$HOME/arm64-libs/lib/libgmp.a \
    -DMPFR_INCLUDE_DIR=$HOME/arm64-libs/include \

### Step 7 — Build GraphBLAS for x86-64

```bash
cd SuiteSparse && mkdir build_x86 && cd build_x86
cmake .. -DCMAKE_BUILD_TYPE=Release \
    -DCMAKE_INSTALL_PREFIX=$HOME/x86-libs \
    -DSUITESPARSE_ENABLE_PROJECTS="graphblas" \
    -DSUITESPARSE_ENABLE_CUDA=OFF
make -j$(nproc) GraphBLAS && make install
cp /usr/lib/x86_64-linux-gnu/libsuitesparseconfig.so* $HOME/x86-libs/lib/
cp /usr/include/suitesparse/SuiteSparse_config.h \
    $HOME/x86-libs/include/suitesparse/
cd ../..
```

### Step 8 — Download test matrix

```bash
cd matrices
wget "https://suitesparse-collection-website.herokuapp.com/MM/Pajek/HEP-th-new.tar.gz"
tar -xzf HEP-th-new.tar.gz && rm HEP-th-new.tar.gz
cd ..
```

## Running benchmarks

```bash
./scripts/run_all.sh
```

Results are saved to `results/` as plain text files named
`<benchmark>_<arch>.txt` (e.g., `cholmod_x86_64.txt`, `klu_riscv64.txt`).

## Environment variables

| Variable | Default | Description |
|---|---|---|
| `RISCV_LIBS` | `$HOME/riscv-libs` | Path to RISC-V SuiteSparse libraries |
| `ARM64_LIBS` | `$HOME/arm64-libs` | Path to AArch64 SuiteSparse libraries |
| `X86_LIBS` | `$HOME/x86-libs` | Path to x86-64 GraphBLAS library |
